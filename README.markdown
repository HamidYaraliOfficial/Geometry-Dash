# Geometry Dash

---

# 🇬🇧 English

## Overview
Geometry Dash is a modern desktop game inspired by the classic Geometry Dash experience, developed with Python and PyQt6.

The game includes:
- Smooth animations and particle effects
- Dynamic obstacle generation
- Double-jump mechanics
- Multiple themes (Light & Dark)
- Multi-language support (English, Persian, Chinese)
- Score, progress, speed, and attempt tracking
- Modern graphical user interface

## Requirements

- Python 3.10 or newer
- PyQt6

## Installation

### Install Required Library

```bash
pip install PyQt6
```

### Run the Game

```bash
python "Geometry Dash.py"
```

## Features

- Real-time gameplay
- Procedural level generation
- Theme switching
- Language switching
- Visual effects and particles
- Progress tracking
- High-score system

## Controls

| Action | Key |
|----------|----------|
| Jump | Space |
| Jump | Left Mouse Click |
| Alternative Jump | W |
| Alternative Jump | Up Arrow |

## Project Structure

```text
Geometry Dash.py
README (1).markdown
```

## Notes

- Make sure PyQt6 is installed before running.
- Recommended screen resolution: 900×600 or higher.
- The game runs on Windows, Linux, and macOS.
